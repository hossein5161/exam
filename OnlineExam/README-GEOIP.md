# راهنمای استفاده از MaxMind GeoIP2

## دانلود فایل GeoLite2-Country.mmdb

برای استفاده از سرویس تشخیص IP، باید فایل پایگاه داده GeoLite2-Country.mmdb را از MaxMind دانلود کنید:

1. به آدرس زیر بروید:
   https://dev.maxmind.com/geoip/geoip2/geolite2/

2. یک حساب کاربری رایگان ایجاد کنید (یا وارد شوید)

3. فایل `GeoLite2-Country.mmdb` را دانلود کنید

4. فایل را در مسیر زیر قرار دهید:
   ```
   OnlineExam/src/main/resources/GeoLite2-Country.mmdb
   ```

## استفاده خودکار

پس از قرار دادن فایل در مسیر بالا، برنامه به صورت خودکار از آن استفاده می‌کند.

## به‌روزرسانی پایگاه داده

پایگاه داده MaxMind به صورت منظم به‌روزرسانی می‌شود. توصیه می‌شود هر ماه فایل را به‌روزرسانی کنید.

## نکات مهم

- فایل GeoLite2-Country.mmdb رایگان است و برای استفاده تجاری نیاز به لایسنس دارد
- برای استفاده تجاری، باید از نسخه پولی GeoIP2 استفاده کنید
- فایل را در Git commit نکنید (به .gitignore اضافه کنید)
